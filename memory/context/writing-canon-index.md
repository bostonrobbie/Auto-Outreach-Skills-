# Writing Canon — master index + mandatory protocol

**This file is the entry point for ALL outreach writing. Load it FIRST, every time.**

Never start outreach writing from scratch or from memory. Before writing anything, load the canon below and write *against* it. After writing, check the draft *against* it.

> Writing is step D of a larger gauntlet. The canon governs the DRAFT. Drafting is only valid AFTER the pre-draft gates (scope/ownership → dedup → persona → live profile read → company web-anchor) and only DONE after the post-draft gates (QC audit → send → MASTER log). Full pipeline: `skills/_shared/outreach-gauntlet.md`. Load it alongside this file.

## Mandatory protocol
**BEFORE:** load canon #1-#3 + the channel formula. Confirm the structure. Research is for RELEVANCE only — never recited at them.
**WHILE:** ask never assert; no resume recap / "I noticed"; no feature-led headline; CTA = engagement question; one named proof; within word count; no em dashes; correct sign-off.
**AFTER:** run the MQS 12-point gate; confirm zero hard-rule violations; apply the "could-anyone-tell" test. Only ≥9/12 drafts go to the rep.
