# Run-my-day prompt — the morning driver

> Paste this each morning to get oriented and set up the day's work.

```
Run my day. In order:
1. Read CLAUDE.md, the gauntlet, and config.json.
2. Scan my inbox for new replies; classify and surface any warm ones (positive / curious /
   referral) with a recommended response. Flag anything time-sensitive.
3. Check my warm-leads and pipeline files for follow-ups owed today.
4. Tell me what's due (T2/T3 follow-ups, callbacks) and any bounces to log to do-not-contact.
5. Recommend the day's priorities and one batch to build.

Surface it as a short briefing, then wait. Draft nothing and send nothing until I pick a task.
```
